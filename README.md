# Sistema de Lista de Presentes: ListaWish

Projeto desenvolvido como Trabalho de Conclusão de Curso.

## Descrição

Este sistema permite que usuários criem listas de presentes e compartilhem um link com convidados para que eles possam visualizar e marcar itens como comprados.

## Tecnologias Utilizadas

- Node.js
- Express
- SQLite
- bcrypt
- Git e GitHub

## Estrutura do Projeto

src/
controllers/
models/
routes/
config/

## Como executar o projeto

1. Clonar o repositório

git clone <https://github.com/odamonique/listawish>

2. Instalar dependências

npm install

3. Rodar o servidor

npm start

O servidor iniciará na porta 3000.
